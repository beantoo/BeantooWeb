Lucid Game Engine

A java game engine that specializes in 2D games. The engine is en-visioned to make simple 2D games and most of its features are yet to be finalized. It will be suited for beginners who want to dabble into 2D game development.

It will be available for Windows and MacOS. A release date is To-Be-Decided.
